### README ###

Movement:
Numpad and WASD

Attacking:
Move into an enemy to attack them

Items:
g - Pick up item
i - Use Item (Opens inventory to select item)
d - Drop Item(Opens inventory to select item)

Select Items in the inventory to equip/dequip them

Skills:
k - Open Skills menu

Escape - Exit Game
L-Alt + Enter - Full Screen Operation
< - Use Stairs
